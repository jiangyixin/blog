## 底部按钮absolute布局，在弹出输入框时也会跟随浮动

采用flex布局解决

```html
<div class="page">
  <div class="herder"></div>
  <div class="container"></div>
  <div class="footer"></div>
</div>
```

```style
.page {
  min-height: 100%;
  display: -webkit-box;
  display: flex;
  flex-direction: column;
}
.herder, .footer {
  flex-shrink: 0;
}
.container {
  flex: 1;
}
```
